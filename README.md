# iRCSoccerB0t
Simple irc stats bot written in python 2.7. 
This bot stores the daily table soccer stats in a mongodb.

# Commands
!stats - show stats

!topic - set topic

# SuperUser Commands
!set $name $value - set specific value for user

!loser [$name]    - +1 for loser

!owner [$name]    - add more owners (su)


# Dependencies
- pymongo
